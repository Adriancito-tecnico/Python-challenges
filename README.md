# ES 🇪🇸
## Desafíos y Ejercicios de Programación en Python
Este repositorio contiene una colección de soluciones a problemas de programación, algoritmos y pequeñas automatizaciones desarrolladas en Python.

El objetivo del proyecto es mantener el código simple, legible y enfocado. En lugar de desarrollar aplicaciones complejas o librerías extensas, cada ejercicio se resuelve en un único archivo ejecutable. Esto permite analizar la lógica de manera aislada, entender la estructura de datos utilizada y ejecutar el código sin necesidad de configurar un entorno pesado.

## Canal de YouTube

Cada script de este repositorio está asociado a un vídeo en YouTube donde se explica el proceso de resolución paso a paso. En cada contenido se aborda:

* El análisis del problema y los casos límite.
* El diseño de la solución lógica antes de escribir código.
* La implementación en Python y su ejecución.

Puedes ver la explicación detallada de cada ejercicio en el canal de YouTube:

# EN 🇬🇧

## Python Coding Challenges and Scripts

This repository contains a collection of solutions to programming problems, algorithmic challenges, and small automation utilities built with Python.

The project focuses on keeping code clear, readable, and self-contained. Rather than building large frameworks or complex applications, each exercise is implemented in a single file. This approach makes it easy to isolate the logic, study the data structures involved, and run the code without setting up a heavy environment.

## YouTube channel

Every script in this repository corresponds to a YouTube video breaking down the solution step by step. Each video covers:

* Problem analysis and potential edge cases.
* Designing the logical solution before writing code.
* The line-by-line implementation in Python and terminal execution.

You can watch the full walk-throughs on the YouTube channel:
